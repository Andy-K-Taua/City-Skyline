# City Skyline Project

A CSS-based cityscape project built to explore layout, positioning, and animation.

## Features
- **Responsive Layout:** Built using CSS variables and flexbox/grid.
- **Interactive Elements:** Features a "sun" aligned to the background gradient and interactive buildings.
- **Animations:** Includes a "destruction" effect for buildings when clicked.

## How to View
1. Clone this repository: 
   `git clone https://github.com/Andy-K-Taua/City-Skyline.git`
2. Open `index.html` in your favorite web browser.

## Live Demo
[View the Live Skyline Project Here](https://andy-k-taua.github.io/City-Skyline/)

## Tech Stack
- HTML5
- CSS3 (Variables, Keyframes, Flexbox)
- JavaScript (DOM Manipulation)
